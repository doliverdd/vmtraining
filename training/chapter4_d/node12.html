Exercises

  1. Write a script that searches through a set of files in a directory and processes the names according to the following rules
    • If the file is a directory print a warning message.
    • If the file is not empty create a file in the results subdirectory of the directory from which the script is written. Do not create the results directory ahead of time, but check to see if it exists and create it if necessary.
    • If the file is empty delete it.
    • keep track of how many of each type of file is encountered.
    Create a directory and some subdirectories, empty files and non-empty files to check your script. Be careful that you don't delete too much. On trick is to write the script with just warning messages first to see if it will do what you think it will.
  2. Check the Pattern-Matching Operators in Appendix and see if you can write a script that will take a PATH variable and print each directory on a line. (Hint: subsitute a \n character for the : delimiter) Can you extend this to a script that will check to see if the directory exists?
  3. Read the name of a file filename of strings from the keyboard. Find the first and last strings alphabetically and put them in a file called first_last. Assume that the file contains one string per line. Check Appendix for string operators. Make up a file and test your script. Try the same thing but get the whole file in alphabetic order by using the sort command.
  4. Advanced Exercise - Command line parameters to the script may be accessed by using $#. The name of the script is located in $0, the next word on the command line is $1 and so on. Write a script that takes three parameters The first is a maximum size. The second, third and fourth parameters are names of files. If the combined size of the file named by the second parameter and the file named by the third parameter is less than the maximum size, then they are combined into the name given in the fourth parameter. The original files are then removed. Can you extend this script so that the last name can match one of the other two?
