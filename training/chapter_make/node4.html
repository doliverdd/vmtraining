Exercises

  1. Add a loop to the main program file in the example that calculates the Bessel function to call the sub-program a large number of times. Experiment with the size of the loop so that the executions takes some significant amount of time. (Say > 3 minutes) Use the Linux "time" command to time your runs.(Check the man page.) Now try compiling the program with different levels of optimization to see if the code gets faster.
  2. Create a makefile so that the code from the Bessel function example can be compiled with make. Extend the file so that if you want to compile with the -g option you make one target, and if you want to compile without the -g option you make another target.
  3. Advanced Exercise - Look up the the -D option for gcc and see if you can use this to compile the program with the Bessel option sometimes and with some other function of your choice at other times. (Even just square root as an example) Put these changes into the makefile so that you can make the version that you want with no changes to the code or makefile.
