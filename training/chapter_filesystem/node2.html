Getting Started

The first thing that you must know is how to access the system. You should have been given an account name and password when you arrived in the lab today. You will need both to access the system since Linux is a true multiuser system. Please keep the account name and password in a safe place. For our examples we will assume that the system has a user with the account name "frodo". Notice that frodo is not capitalized. This is because Linux is case sensitive. Before we log on to the system let's take a look at the filesystem structure.

Figure 2.1: Linux File System

The filesystem on Linux is composed of directories and files. The directories are synonymous with folders in the Windows operating systems. The main system directories are displayed in Figure 2.1. On most Linux systems the contents of these main directories are fairly standard, although the reasons may not be clear. The common contents of some of the more important directories for users are listed in Table 2.1.


Table 2.1: Linux System Directories
Directory Contents
/bin Command binaries
/home Users home directories
/lib System libraries
/sbin System binaries
/tmp Scratch Space
/usr/bin Binaries for user commands
/usr/include Header or include files
/usr/lib Libraries
/usr/local Locally installed
