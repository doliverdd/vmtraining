if ... then ... else

No language would be viable without some manner of conditional control of flow. In bash this construction takes the form below. 
if [ ... ]
then
  # if-code
else
  # else-code
fi

So lets change the script that creates a directory for each data file to check and see if the directory exists. 

#!/bin/bash
# create a directory for each data file

for dir in `ls data` ; do
   if [ -e results/${dir} ]
   then
      echo The file/directory results/${dir} exists
   else
      mkdir results/$dir
   fi
done

This example also introduces the file attribute operator. We alluded to this before when discussing the test operator. In this case we are using the operator -e which test to see if a file exists. If the file does exist we print a message, and if not we create the directory. A listing of the other file attribute operators can be found in Appendix .
