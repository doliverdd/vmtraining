Working with the Command Line Environment

There are many points of view on working with a command line operating system. And for each point of view there are pluses and minuses for both the command line and GUI environments. In this chapter we will explore two of the advantages of this environment. The first is that the configuration of the environment is very flexible. This is accomplished by setting the shell environment variables. The other advatage we will be looking at in this environment is the use of shell scripts. These are small easy to write programs that allow you to do repetative tasks by calling a simple command. To accomplish these tasks we must introduce and editor so that you can maniuplate the contents of files. There is an editor on all *nix systems called vi. In the next section we will introduce you to vi through a series of practice examples.
