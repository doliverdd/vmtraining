Part command

  • The previous slide has been created by the `1#1part' command;
  • This command does generates a slide but not a table of contents section;
  • This command has the following optional arguments:
    • toc: overwrites toc entry;
    • trans: specific transition effect;
    • template: slide template that should be used to create the slide. This template should be supported as a regular slide template by the style that you use for your presentation.
  • Regular slides also support the first two options.

Sectionandpart command

  • The section in the table of contents called `Section 2' has been created by the `1#1sectionandpart' command;
  • This command generates a slide and a table of contents section;
  • This command has the same optional arguments as the `1#1part' command.

[toc=]Notes

  • This slide contains notes.
  • HA-prosper package options for notes:
    • slidesonly (default): produces only slides. Recommended for the actual presentation.
    • notesonly: produces only the notes.
    • notes: includes the notes into the presentation for printing purposes or for distributing the entire presentation.
  • If you want to print only the notes, first run the presentation with the option `notes' to create page number labels, then run it with the option `notesonly'.
