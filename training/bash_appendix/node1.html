BASH REFERENCE

File Attribute Operators
-d file file exists and is a directory
-e file file exists
-f file file exists and is a regular file
-r file You have read permissions on the file
-s file file exists and is not empty
-w file You have write permissions on the file
-x file You have execute permissions on the file
-O file You own the file
-G file Files group id matches one of yours
file1 -nt file2 file1 is newer than file2
file1 -ot file2 file1 is older than file2
Arithmetic Test Operators
-lt Less than
-le Less than or equal
-eq Equal
-ge Greater than or equal
-gt Greater than
-ne Not equal

Relational Operators
< Less than
> Greater than
<= Less than or equal to
>= Greater than or equal to
== Equal to
!= Not equal to
String Comparison Operators
str1 = str2 str1 equals str2
str1 != str2 str1 does not equal str2
str1 < str2 str1 is less than str2
str1 > str2 str1 is greater than str2
-n str1 str1 is not null
-z str1 str1 is null
Arithmetic Operators
+ Plus
- Minus
* Multiplication
/ Division
% Remainder
<< Bit-shift left
>> Bit-shift right
& Bitwise and
| Bitwise or
~ Bitwise not
! Bitwise not
^ Bitwise exclusive or
Pattern-Matching Operators
${variable#pattern} If the pattern matches the
beginning of the variables value, delete the shortest part
that matches and return the rest.
${variable##pattern} If the pattern matches the
beginning of the variables value, delete the longest part
that matches and return the rest.
${variable%pattern} If the pattern matches the
end of the variables value, delete the shortest part
that matches and return the rest.
${variable%%pattern} If the pattern matches the
end of the variables value, delete the longest part
that matches and return the rest.
${variable/pattern/string}  
${variable//pattern/string} The longest match
to pattern in variable is replaced by string.
In the first form only the first match replaced.
In the second form all matches are replaced. If
The pattern begins with # the match must
be at the begining. % means the match
must be at the end. If the string is null
then the matches are deleted.
